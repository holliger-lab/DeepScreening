# DeepScreening

This is the source code accompaniment to the Deep Screening [paper]().

It contains the:
- Analysis pipeline for processing a deep screening dataset.
- Training code for the Bert-DS language model.

## Core dependencies

- numpy
- scipy
- MPI4py
- scikit-image
- opencv
- pytorch
- scikit-learn
- zeromq
- matplotlib
- pytorch_lightning
- torchmetrics
